Stable Sorting Algorithms

Bubble Sort

Insertion Sort

Merge Sort

Example
Input
(1,A) (1,B) (2,C)

(1,A) (1,B) (2,C)

Order preserved.

Unstable Sorting Algorithms

Selection Sort

Quick Sort

Heap Sort
